# Basics
- The `count()` method returns the number of times a specified value appears in the string. [W3](https://www.w3schools.com/python/ref_string_count.asp)
```python
txt = "I love apples, apple are my favorite fruit"
x = txt.count("apple")
print(x)
# output 2
```

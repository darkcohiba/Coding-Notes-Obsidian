# Basics
- Make the lower case letters upper case and the upper case letters lower case [W3](https://www.w3schools.com/python/ref_string_swapcase.asp)
```python
txt = "Hello My Name Is PETER"

x = txt.swapcase()

print(x)
# output hELLO mY nAME iS peter
```

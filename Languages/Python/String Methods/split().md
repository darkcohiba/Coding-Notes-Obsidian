# Basics
- Split a string into a list where each word is a list item. [W3](https://www.w3schools.com/python/ref_string_split.asp)
```python
txt = "welcome to the jungle"

x = txt.split()

print(x)
#output ['welcome', 'to', 'the', 'jungle']
```

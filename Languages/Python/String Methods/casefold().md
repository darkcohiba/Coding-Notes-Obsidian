# Basics
- The `casefold()` method returns a string where all the characters are lower case. [W3](https://www.w3schools.com/python/ref_string_casefold.asp)
```python
txt = "Hello, And Welcome To My World!"  
x = txt.casefold()  
print(x)
# hello, and welcome to my world!
```

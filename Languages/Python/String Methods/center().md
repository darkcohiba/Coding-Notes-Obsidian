# Basics
- The `center()` method will center align the string, using a specified character (space is default) as the fill character. [W3](https://www.w3schools.com/python/ref_string_center.asp)
```PYTHON
txt = "banana"  
x = txt.center(20, "O")  
print(x)
# OOOOOOObananaOOOOOOO
```

# Basics
- The `find()` method finds the first occurrence of the specified value.
- The `find()` method returns -1 if the value is not found. [W3](https://www.w3schools.com/python/ref_string_find.asp)
```python
txt = "Hello, welcome to my world."
x = txt.find("welcome")
print(x)
# output 7
```

# Basics
- The `endswith()` method returns True if the string ends with the specified value, otherwise False. [W3](https://www.w3schools.com/python/ref_string_endswith.asp)
```python
txt = "Hello, welcome to my world."

x = txt.endswith(".")

print(x)
# output True
```

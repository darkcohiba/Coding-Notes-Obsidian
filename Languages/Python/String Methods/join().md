# Basics
- Join all items in a tuple into a string, using a hash character as separator [W3](https://www.w3schools.com/python/ref_string_join.asp)
```python
myTuple = ("John", "Peter", "Vicky")

x = "#".join(myTuple)

print(x)
# output John#Peter#Vicky
```

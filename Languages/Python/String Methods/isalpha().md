# Basics
- The `isalpha()` method returns True if all the characters are alphabet letters (a-z).
- Example of characters that are not alphabet letters: (space)!#%&? etc. [W3](https://www.w3schools.com/python/ref_string_isalpha.asp)
```python
txt = "CompanasdfyX"
x = txt.isalpha()
print(x)
# output True
```
